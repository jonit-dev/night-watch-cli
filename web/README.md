# Night Watch Web Dashboard

The web UI for Night Watch — a dashboard for monitoring and managing PRDs, pull requests, automation schedules, and project settings.

## Getting Started

**Prerequisites:** Node.js

1. Install dependencies:
   ```bash
   yarn install
   ```
2. Start the development server:
   ```bash
   yarn dev
   ```

The dashboard will be available at `http://localhost:5173` (default Vite port).
